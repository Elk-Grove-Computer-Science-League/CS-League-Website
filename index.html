We’re Elk Grove CS League, a program where students teach students how to code

Ever wanted to try coding, or thinking about majoring in computer science in college? CS League is a great way to learn - we cover everything from website design to Python, all from a beginner standpoint.

Here's how it works

Events

Students-run weekly coding courses on Zoom, with rotating topics

Hackathon

In-person meetups with crash courses and competiitions, four times a year

Community

Game nights, coding support, and voice chat hangouts on Discord

Learn anything

The cool part about CS League is that we cover a huge range of CS topics - you get to sign up for the events and hackathons you’re interested in. These are a few of the topics we teach:

  • Web apps
  • Website design (UI/UX)
  • JavaScript
  • Python
  • Cloud Computing
  • Machine Learning
  • Game dev (coming soon!)

Ready to join?

Raffles, team games, free pizza, and coding music videos (yes, music videos) - you won't wanna miss out!
